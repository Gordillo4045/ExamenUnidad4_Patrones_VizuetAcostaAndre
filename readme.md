# ExamenUnidad4_Patrones_VizuetAcostaAndre

## Vizuet Acosta Andre

## Gestión de entregas de comercio electrónico 

## Descripción General 

Este es un sistema de gestión de pedidos de comercio electrónico que implementa 4 patrones de diseño. El sistema maneja el ciclo completo de vida de un pedido: creación, cambios de estado (pago, envío, entrega), cancelación.

### Ejecución 

`Paso 1`: Descargar .NET 9.0 SDK desde https://dotnet.microsoft.com/download y git https://git-scm.com/downloads

`Paso 2`: Clonar el repositorio en **Visual Studio Code**.
```
git clone https://github.com/Gordillo4045/ExamenUnidad4_Patrones_VizuetAcostaAndre
```
`Paso 3`: Abrir terminal y navegar a la carpeta del proyecto
```
cd ExamenUnidad4_Patrones_VizuetAcostaAndre/src
```
`Paso 4`: Compilar y ejecutar el proyecto
```
dotnet build
dotnet run
```
